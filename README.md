# HTML Roadmap

## 1️⃣ Basics

Learn the structure and basic building blocks of a webpage:

* `<!DOCTYPE html>`
* `<html>`, `<head>`, `<body>`
* `<title>`, `<meta>`
* Headings: `<h1>` → `<h6>`
* Paragraphs: `<p>`
* Line breaks: `<br>`
* Horizontal lines: `<hr>`
* Comments: `<!-- comment -->`

---

## 2️⃣ Text & Formatting

* Bold, italic, underline: `<b>`, `<strong>`, `<i>`, `<em>`
* Subscript & superscript: `<sub>`, `<sup>`
* Blockquote: `<blockquote>`
* Code text: `<code>`
* Preformatted: `<pre>`

---

## 3️⃣ Links & Images

* Links: `<a href="URL">text</a>`
* Target attribute: `_blank`, `_self`
* Images: `<img src="url" alt="description">`
* Image attributes: width, height, alt
* Favicon for your site


---

## 4️⃣ Lists

* Ordered list: `<ol>` + `<li>`
* Unordered list: `<ul>` + `<li>`
* Nested lists

---

## 5️⃣ Tables

* Table: `<table>`
* Rows: `<tr>`
* Cells: `<td>` (data), `<th>` (header)
* Attributes: colspan, rowspan, border
* Caption: `<caption>`

---

## 6️⃣ Forms & Inputs

* Form tag: `<form>`
* Inputs: text, email, password, number, checkbox, radio
* Labels: `<label>`
* Textarea: `<textarea>`
* Buttons: `<button>`
* Select dropdown: `<select>` + `<option>`
* Form attributes: action, method, placeholder, required

---

## 7️⃣ Semantic HTML

* Header: `<header>`
* Navigation: `<nav>`
* Main content: `<main>`
* Section: `<section>`
* Article: `<article>`
* Footer: `<footer>`
* Aside: `<aside>`
* Figure & caption: `<figure>` + `<figcaption>`

---

## 8️⃣ Multimedia & Embeds

* Audio: `<audio controls>`
* Video: `<video controls>`
* Embed YouTube: `<iframe>`
* Object/embed content: `<object>`, `<embed>`

---

## 9️⃣ Advanced Attributes & Meta

* Meta tags: charset, viewport, description, keywords
* Global attributes: `id`, `class`, `style`, `title`, `hidden`
* Data attributes: `data-*`
* Accessibility: `alt`, `aria-label`, `role`
